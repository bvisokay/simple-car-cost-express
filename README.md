### Overview

:blue_car: Users can calculate a simple "cost per remaining month" for a given vehicle given based on how much driving they do, expected life of a car, current mileage, and price. Users can save links to online vehicle listings from across the web in one place.

:traffic_light: Note: This is a deprecated version of a live project

### Stack

:computer: Node, Express, MongDB, EJS, Vanilla JS, Boostrap

### Features List

- :heavy_check_mark: Local storage for unauthenticated users
- :heavy_check_mark: Authentication for user-specific data persistance
- :heavy_check_mark: Create-Read-Update-Delete functionality with MongoDB
- :heavy_check_mark: Authenticated users can customize default settings
- :heavy_check_mark: Filtering items for authenticated users
